# eightPuzzle_BFS_IDDFS
Simulador do jogo 8-Puzzle desenvolvido em C, que permite ao usuário jogar manualmente ou visualizar a solução automática utilizando algoritmos de Inteligência Artificial. O projeto implementa Busca em Largura (BFS) e Busca em Profundidade Limitada Iterativa (IDDFS) utilizando estruturas de dados como filas e pilhas.
